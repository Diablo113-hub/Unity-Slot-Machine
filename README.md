# Unity Slot Machine

## Overview

This is a simple slot machine game developed in Unity as part of a game development assignment.

The game features randomized slot spinning, winning combinations, payout logic, reel animations, and a WebGL build.

---

## Features

* Randomized slot results using RNG
* Smooth reel spin animations
* Jackpot win condition
* Coin score system
* Different winning symbol combinations
* WebGL playable build

---

## Winning Logic

The player wins when all three slot symbols match.

Example:

* 777 = Jackpot
* Cherry Cherry Cherry = Cherry Win
* Bell Bell Bell = Bell Win
* Bar Bar Bar = Bar Win

---

## Controls

* Press the lever/spin button to spin the reels.
* Coins are deducted per spin.
* Winning combinations reward coins.
